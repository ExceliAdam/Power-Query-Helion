# Power Query w Excelu - Analizuj dane jak profesjonalista

Materiały dodatkowe do książki **"Power Query w Excelu - Analizuj dane jak profesjonalista"** autorstwa Adama Kopcia (Miłośnik Excela).

## O książce

Poradnik o pracy z Power Query w Excelu. Znajdziesz tu wszystkie pliki potrzebne do samodzielnego wykonania ćwiczeń z książki.

## Zawartość repo

- **pliki-poczatkowe/** - pliki startowe do ćwiczeń (tu zaczynasz)
- **pliki-koncowe/** - gotowe rozwiązania (tu sprawdzasz wyniki)
- **rysunki/** - ilustracje z książki
- **kod-m/** - pliki z kodem M

## Jak korzystać

1. Pobierz pliki z folderu **pliki-poczatkowe** dla danego rozdziału
2. Wykonaj ćwiczenia zgodnie z instrukcjami w książce
3. Porównaj swój wynik z plikami z folderu **pliki-koncowe**
