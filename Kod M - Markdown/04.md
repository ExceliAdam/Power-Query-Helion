# 4. Transformacje tekstu

## Wyodrębnianie tekstu

Przed ogranicznikiem:
```powerquery
= Table.AddColumn(#"Zmieniono typ", "Tekst przed ogranicznikiem", each Text.BeforeDelimiter([Dane], ","), type text)
```

Zmiana nazwy nowej kolumny w przekształceniu wyciągającym dane przed ogranicznikiem:
```powerquery
= Table.AddColumn(#"Zmieniono typ", "Imię i Nazwisko", each Text.BeforeDelimiter([Dane], ","), type text)
```

Tekst między ogracznikami:
```powerquery
= Table.TransformColumns(#"Wstawiony tekst przed ogranicznikiem[WŻ10.1][AK10.2]", {{"Dane", each Text.BetweenDelimiters(_, "; ", "|", {0, RelativePosition.FromEnd}, 0), type text}})
```

Zmiana kolejności kolumn:
```powerquery
= Table.ReorderColumns(#"Zmieniono typ1",{"Imię i Nazwisko", "Wiek"})
```

## Scalanie tekstu

Wyciąganie pierwszego znaku:
```powerquery
= Table.AddColumn(#"Zamienione błędy", "Pierwsze znaki", each Text.Start([Imię], 1), type text)
```

Wyciąganie pierwszego znaku z dołączaniem kropki:
```powerquery
= Table.AddColumn(#"Zamienione błędy", "Inicjał imienia", each Text.Start([Imię], 1) & ".", type text)
```

Proste scalanie różnych wartości w kolumnie niestandardowej:
```powerquery
Text.Start([Imię], 1) & "." & " " & Text.Start([Nazwisko], 1) & "."
```

Ręczne scalanie dwóch kolumn tekstowych z użyciem łącznika (spacji):
```powerquery
= Table.AddColumn(#"Zmieniono typ", "Imię i nazwisko", each [Imię] & " " & [Nazwisko], type text)
```

Dodanie kolumny ze stałą wartością:
```powerquery
= Table.AddColumn(#"Wstawiono pierwsze znaki", "Miasto", each "Wąchock", type text)
```

Ręczne scalanie w kolumnie niestandardowej:
```powerquery
[Imię] & "," & [Drugie] & ", " & [Nazwisko]
```

Scalanie w pełnej formule z funkcją Text.Combine:
```powerquery
= Table.AddColumn(#"Dodano kolumnę niestandardową", "Scalone", each Text.Combine({[Imię], [Drugie], [Nazwisko]}, ", "), type text)
```

Scalanie z funckją Combiner:
```powerquery
= Table.CombineColumns(#"Wstawiono scaloną kolumnę",{"Nazwisko", "Drugie", "Imię"},Combiner.CombineTextByDelimiter(", ", QuoteStyle.None),"Scalone.1")
```

Scalanie z własną funkcją:
```powerquery
= Table.CombineColumns(#"Wstawiono scaloną kolumnę",{"Imię", "Drugie", "Nazwisko"},(x) => Text.Combine(List.RemoveItems((x), {null, ""}),", "),"Scalone.1")
```

## Podział tekstu

Podział na domyślne kolumny:
```powerquery
= Table.SplitColumn(#"Zmieniono typ", "Polecane osoby", Splitter.SplitTextByDelimiter(", ", QuoteStyle.Csv), {"Polecane osoby.1", "Polecane osoby.2", "Polecane osoby.3", "Polecane osoby.4"})
```

Podział na określoną liczbę kolumn:
```powerquery
= Table.SplitColumn(#"Zmieniono typ", "Polecane osoby", Splitter.SplitTextByDelimiter(", ", QuoteStyle.Csv), 5)
```

Podział bez określenia liczby kolumn:
```powerquery
= Table.SplitColumn(#"Zmieniono typ", "Polecane osoby", Splitter.SplitTextByDelimiter(", ", QuoteStyle.Csv))
```

## Formatowanie i czyszczenie tekstu

Usuwanie zbędnych spacji z początku i końca tekstu (Przycięcie):
```powerquery
= Table.TransformColumns(#"Podzielono kolumnę", {{"Miasto", Text.Trim, type text}})
```

Zmiana wielkości liter:
* **Wielkie litery:** `Text.Upper([Kraj])`
* **Małe litery:** `Text.Lower([Kraj])`
* **Każdy wyraz z wielkiej litery:** `Text.Proper([Kraj])`


## Podział według pozycji

Podział według pozycji - domyślne nazwy kolumn:
```powerquery
= Table.SplitColumn(#"Zmieniono typ", "Dane", Splitter.SplitTextByPositions({0, 10, 21, 32}), {"Dane.1", "Dane.2", "Dane.3", "Dane.4"})
```

Podział według pozycji - przypisane ręcznie nazwy kolumn:
```powerquery
= Table.SplitColumn(#"Zmieniono typ", "Dane", Splitter.SplitTextByPositions({0, 10, 21, 32}), {"Miasto", "Przychód", "Koszty", "Produkt"})
```

Podział przy przejściu ze znaku na cyfrę:
```powerquery
= Table.SplitColumn(#"Zmieniono typ1", "Produkt", Splitter.SplitTextByCharacterTransition((c) => not List.Contains({"0".."9"}, c), {"0".."9"}), {"Produkt.1", "Produkt.2"})
```
