# 3. Złożone źródła danych

## Pobieranie danych z arkusza lub tabeli

Odwołanie do zawartości konkretnego obiektu (arkusza lub tabeli) w pliku Excel:
```powerquery
= Source{[Item="Sprzedaż",Kind="Sheet"]}[Data]
```

Rozwinięcie scalonej kolumny w celu wyciągnięcia konkretnych pól:
```powerquery
= Table.ExpandRecordColumn(_Sprzedaż, "Produkty", {"Cena zł"}, {"Cena zł"})
```

Rozwinięcie z jednoczesną zmianą nazwy kolumny:
```powerquery
= Table.ExpandRecordColumn(_Sprzedaż, "Produkty", {"Cena zł"}, {"Cena"})
```

## Operacje na wierszach i kolumnach

Wstawienie mnożenia ze wstążki:
```powerquery
= Table.AddColumn(#"Rozwinięty element Produkty", "Mnożenie", each [Masa] * [Cena], Currency.Type)
```

Wstawienie mnożenia ze wstążki z ręcznie wpisaną nazwą nowej kolumny:
```powerquery
= Table.AddColumn(#"Rozwinięty element Produkty", "Przychód", each [Masa] * [Cena], Currency.Type)
```

Usuwanie duplikatów z zaznaczonych kolumn:
```powerquery
= Table.Distinct(#"Usunięto inne kolumny", {"Produkt", "Odmiana"})
```

Odwracanie kolejności wierszy w tabeli:
```powerquery
= Table.ReverseRows(#"Zmieniono typ")
```
