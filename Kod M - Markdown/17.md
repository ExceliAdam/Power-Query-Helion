# 17. Optymalizacja i dobre praktyki podczas tworzenia zapytań

## Optymalizacja

Zmiana domyślnej formuły grupowania na połączenie dwóch funkcji w celu uzyskania listy najczęstszych produktów:

```powerquery
each Text.Combine(List.Modes([Produkt]), ", ")
```

## Unikanie błędów

Pobieranie zawartości tabeli z bieżącego skoroszytu na podstawie jej nazwy (zamiast indeksu):

```powerquery
= Excel.CurrentWorkbook(){[Name=WybranaTabela]}[Content]
```

Odwoływanie się do konkretnego rekordu w tabeli parametrów poprzez nazwę klucza (np. dla konsultacji):

```powerquery
{[Name="Konsultacje2023"]}
```

Przykład nazewnictwa kroku bez spacji, co upraszcza strukturę kodu i późniejsze odwołania:

```powerquery
DodanieWeryfikacjiPESEL
```

## Automatyczne odświeżanie zapytań

Makro VBA zarejestrowane w celu odświeżania zapytania na podstawie zaznaczenia, odświeżania wszystkiego oraz odświeżania konkretnego połączenia:

```vba
Sub Odświeżanie()
' Odświeżanie Makro
Range("D3").Select
Selection.ListObject.QueryTable.Refresh BackgroundQuery:=False
ActiveWorkbook.RefreshAll
ActiveWorkbook.Connections("Zapytanie — Rozdział16").Refresh
End Sub
```

## Czas wykonywania zapytania

Makro VBA mierzące i wyświetlające czas odświeżania zapytania, w którego obrębie znajduje się zaznaczona komórka:

```vba
Sub CzasOdświeżenia()
Dim t0 As Double
t0 = Timer
Selection.ListObject.QueryTable.Refresh BackgroundQuery:=False
MsgBox "Czas trwania: " & Timer - t0 & " sekund"
End Sub
```
