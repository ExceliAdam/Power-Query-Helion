# 2. Podstawowe źródła danych i przekształcenia

## Pobieranie danych ze strony WWW

Pobieranie tabeli ze strony internetowej:
```powerquery
= Web.Page(Web.Contents("https://exceliadam.pl/powiaty"))
```

Wyodrębnienie konkretnej tabeli z kodu HTML:
```powerquery
= Source{0}[Data]
```

## Przekształcanie danych tekstowych

Usuwanie zbędnych spacji z początku i końca tekstu (Przycięcie) dla wszystkich kolumn:
```powerquery
= Table.TransformColumns(#"Poprzedni krok", {{"Powiat", Text.Trim, type text}, {"Województwo", Text.Trim, type text}})
```

Podniesienie pierwszego wiersza do rangi nagłówków:
```powerquery
= Table.PromoteHeaders(#"Poprzedni krok", [PromoteAllScalars=true])
```

## Import z plików tekstowych i CSV

Pobieranie danych z pliku tekstowego (rozdzielanego):
```powerquery
= Csv.Document(File.Contents("C:\Ścieżka\Dane.txt"),[Delimiter=" ", Columns=3, Encoding=1250, QuoteStyle=QuoteStyle.None])
```

## Zmiana typów danych

Ręczna zmiana typu danych dla kolumn:
```powerquery
= Table.TransformColumnTypes(#"Nagłówki o podwyższonym poziomie", {{"Powiat", type text}, {"Ludność", Int64.Type}, {"Powierzchnia", type number}})
```
