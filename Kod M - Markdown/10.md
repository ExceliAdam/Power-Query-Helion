# 10. Rankingi i porównywanie wierszy

## Ranking sprzedawców

Sortowanie:
```powerquery
= Table.Sort(#"Zmieniono typ",{{"Obrót", Order.Descending}})
```

Dodanie kolumny indeksu (miejsce w rankingu):
```powerquery
= Table.AddIndexColumn(#"Posortowano wiersze", "Indeks", 1, 1, Int64.Type)
```

Grupowanie danych według obrotu w celu obsługi remisów:
```powerquery
= Table.Group(#"Zmieniono typ", {"Obrót"}, {{"Liczba", each Table.RowCount(_), Int64.Type}})
```

Grupowanie z wieloma sposobami agragacji danych:
```powerquery
= Table.Group(#"Dodano indeks", {"Obrót"}, {{"Tabela", each _, type table [Id pracownika=nullable number, Obrót=nullable number, Ranking=number]}, {"Ranking Śr", each List.Average([Ranking]), type number}, {"Ranking Min", each List.Min([Ranking]), type number}})
```

## Porównywanie danych między wierszami

Scalanie zapytania z samym sobą (przesunięcie wierszy):
```powerquery
= Table.NestedJoin(#"Dodano indeks1", {"Indeks"}, #"Dodano indeks1", {"Indeks.1"}, "Dodano indeks1", JoinKind.LeftOuter)
```

Wyliczanie różnicy (zmiany stanu) między kolejnymi wierszami:
```powerquery
if [Produkt] = [Produkt.1] then [Stan] - [Stan.1] else "początek"
```

## Procentowa zmiana bilansu

Wyszczególnienie wartości (skalara) z konkretnego wiersza i kolumny:
```powerquery
= #"Zmieniono typ"{2}[Bilans]
```

Schemat wyszczególnienia:
```powerquery
#"Nazwa kroku (tabeli)" {indeks (numer) wiersza} [Nazwa kolumny]
```

Wyszczególnienie kolumny (listy):
```powerquery
= #"Zmieniono typ"[Bilans]
```

Wyszczególnienie wiersza (rekordu):
```powerquery
= #"Zmieniono typ"{2}
```

Obliczanie procentowej zmiany względem wartości z wcześniejszego wiersza:
```powerquery
1 - #"Dodano indeks"{[Indeks]}[Bilans]/[Bilans]
```

## Ranking wewnątrz grup

Wynik grupowania wszystkich wierszy:
```powerquery
= Table.Group(#"Zmieniono typ", {"Dyscyplina"}, {{"Tablica", each _, type table [Zawodnik=nullable text, Dyscyplina=nullable text, Punkty=nullable number]}})
```

Wynik grupowania wszystkich wierszy bez szczegółów tabeli:
```powerquery
= Table.Group(#"Zmieniono typ", {"Dyscyplina"}, {{"Tablica", each Table.Sort(_, {"Punkty", Order.Descending}), type table}})
```

Dodanie kolumny indeksu (rankingu) wewnątrz każdej zgrupowanej tabeli:
```powerquery
= Table.Group(#"Zmieniono typ", {"Dyscyplina"}, {{"Tablica", each Table.AddIndexColumn(Table.Sort(_, {"Punkty", Order.Descending}), "Miejsce", 1, 1), type table}})
```
