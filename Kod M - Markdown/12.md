# 12. Parametry i parametryzacja zapytań

## Wybieranie tabeli do zapytania

Referencja do nazwy tabeli w kroku Źródło:

```powerquery
= Excel.CurrentWorkbook(){[Name="Konsultacje2023"]}[Content]
```

Wyszukiwanie indeksu wiersza za pomocą testu logicznego:

```powerquery
Źródło{[Name="Konsultacje2023"]}[Content]
```

Podstawienie parametru `WybranaTabela` w miejsce nazwy tabeli:

```powerquery
= Excel.CurrentWorkbook(){[Name=WybranaTabela]}[Content]
```

Odwołanie do parametru zawierającego znaki specjalne w nazwie:

```powerquery
#"Wybrana Tabela"
```

## Parametr jako filtr

Filtrowanie danych w kolumnie na podstawie parametru o tej samej nazwie:

```powerquery
= Table.SelectRows(#"Zmieniono typ", each [Dział] = Dział)
```

Filtrowanie na podstawie parametru `DziałZapytanie` pobierającego dane z wyniku innego zapytania:

```powerquery
= Table.SelectRows(#"Zmieniono typ", each [Dział] = DziałZapytanie)
```

## Parametry z komórek Excela

Kod M z wartościami dat wpisanymi na stałe (punkt wyjścia do parametryzacji):

```powerquery
= Table.SelectRows(#"Zmieniono typ", each [Data rozmowy] >= #date(2023, 11, 1) and [Data rozmowy] <= #date(2024, 2, 28))
```

Wyszczególnienie wartości z komórki wewnątrz bieżącego zapytania:

```powerquery
= #"Zmieniono typ"{0}[Wartość]
```

Odwołanie do wartości z innego zapytania (`Parametry`) na podstawie indeksu wiersza:

```powerquery
Parametry{0}[Wartość]
```

Odwołanie do wartości z zapytania `Parametry` przy użyciu klucza tekstowego:

```powerquery
Parametry{[Opcja="Data Start"]}[Wartość]
```

Konwersja pobranej wartości na typ daty:

```powerquery
Date.From(Parametry{0}[Wartość])
```

Formuła filtrująca wykorzystująca dynamiczne daty pobierane i konwertowane z zapytania `Parametry`:

```powerquery
= Table.SelectRows(#"Zmieniono typ", each [Data rozmowy] >= Date.From(Parametry{0}[Wartość]) and [Data rozmowy] <= Date.From(Parametry{1}[Wartość]))
```

## Referencje do komórek na podstawie kolumn

Odwołanie do parametru na podstawie nazwy kolumny (po transpozycji tabeli):

```powerquery
ParametryKolumny{0}[Data Start]
```

Formuła filtrująca wykorzystująca nową strukturę tabeli parametrów:

```powerquery
= Table.SelectRows(#"Zmieniono typ", each [Data rozmowy] >= ParametryKolumny{0}[Data Start] and [Data rozmowy] <= ParametryKolumny{0}[Data Koniec])
```
