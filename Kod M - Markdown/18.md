# 18. Power Query w Power BI Desktop

## Przykład 1: Importowanie danych z Excela

Opcja importu zapytań, modeli danych (Power Pivot) oraz Power View z istniejącego pliku Excela:

```text
Plik -> Importuj -> Power Query, Power Pivot, Power View
```

Zmiana statusu ładowania zapytania do modelu danych (opcja „Włącz ładowanie”):

```text
Prawy przycisk myszy na zapytaniu -> Włącz ładowanie
```

Zastosowanie zmian ustawień zapytań bez zamykania edytora:

```text
Zamknij i zastosuj -> Zastosuj
```

## Przykład 2: Diagnostyka w Power BI Desktop

Włączanie szczegółowych opcji śledzenia i zbierania danych wydajnościowych:

```text
Opcje i ustawienia -> Opcje -> Diagnostyka -> Włącz śledzenie / Zagregowane / Szczegółowe / Liczniki wydajności
```

Uruchomienie rejestrowania operacji w celu analizy wydajności kroków:

```text
Karta Narzędzia -> Rozpocznij diagnostykę
```

Analiza czasu trwania i szczegółów wykonania dla konkretnego kroku zapytania:

```text
Karta Narzędzia -> Diagnozuj krok
```

Zatrzymanie procesu diagnostyki i wygenerowanie zapytań z wynikami (Counters, Detailed, Aggregated, Partitions):

```text
Karta Narzędzia -> Zatrzymaj diagnostykę
```
