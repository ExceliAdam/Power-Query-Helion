# 15. Tworzenie funkcji

## Przykład 1: Walidacja numeru PESEL

Wpisanie testowego numeru PESEL jako tekstu (krok początkowy):

```powerquery
="00210155875"
```

Sprawdzenie, czy PESEL składa się dokładnie z 11 znaków:

```powerquery
#"Czy Poprawna Długość" = Text.Length(PESEL) = 11
```

Weryfikacja, czy podany ciąg znaków jest liczbą (z obsługą błędów):

```powerquery
= try Number.From(PESEL) is number otherwise false
```

Zamiana tekstu PESEL na listę pojedynczych cyfr skonwertowanych na liczby:

```powerquery
Cyfry = List.Transform(Text.ToList(PESEL), Number.From)
```

Zdefiniowanie listy wag dla kolejnych cyfr numeru PESEL:

```powerquery
Wagi = {1, 3, 7, 9, 1, 3, 7, 9, 1, 3}
```

Przemnożenie pierwszych 10 cyfr numeru PESEL przez odpowiadające im wagi:

```powerquery
Iloczyny = List.Transform({0..9}, each Cyfry{_} * Wagi{_})
```

Obliczenie sumy wszystkich przemnożonych cyfr:

```powerquery
SumaIloczynów = List.Sum(Iloczyny)
```

Wyliczenie cyfry kontrolnej na podstawie reszty z dzielenia sumy przez 10:

```powerquery
WyliczonaCyfraKontrolna = Number.Mod(10 - Number.Mod(SumaIloczynów, 10), 10)
```

Wyciągnięcie 11. cyfry (kontrolnej) z oryginalnego numeru PESEL:

```powerquery
CyfraKontrolnaZPesel = Cyfry{10}
```

Końcowy warunek logiczny sprawdzający poprawność całego numeru:

```powerquery
CzyPoprawny = if #"Czy Poprawna Długość" and CzyLiczba then WyliczonaCyfraKontrolna = CyfraKontrolnaZPesel else false
```

Pełny kod funkcji niestandardowej `SprawdźPESEL`:

```powerquery
(PESEL as text) as logical => 
let 
    #"Czy Poprawna Długość" = Text.Length(PESEL) = 11, 
    CzyLiczba = try Number.From(PESEL) is number otherwise false, 
    Cyfry = List.Transform(Text.ToList(PESEL), Number.From), 
    Wagi = {1, 3, 7, 9, 1, 3, 7, 9, 1, 3}, 
    Iloczyny = List.Transform({0..9}, each Cyfry{_} * Wagi{_}), 
    SumaIloczynów = List.Sum(Iloczyny), 
    WyliczonaCyfraKontrolna = Number.Mod(10 - Number.Mod(SumaIloczynów, 10), 10), 
    CyfraKontrolnaZPesel = Cyfry{10}, 
    CzyPoprawny = if #"Czy Poprawna Długość" and CzyLiczba then WyliczonaCyfraKontrolna = CyfraKontrolnaZPesel else false 
in 
    CzyPoprawny
```

Zoptymalizowana wersja funkcji (przyspiesza działanie przy błędnych danych):

```powerquery
(PESEL as text) as logical => 
let 
    ListaCyfr = if Text.Length(PESEL) = 11 and (try Number.From(PESEL) is number otherwise false) then List.Transform(Text.ToList(PESEL), Number.From) else null, 
    Wynik = if ListaCyfr = null then false else Number.Mod( 10 - Number.Mod( List.Sum(List.Transform({0..9}, each ListaCyfr{_} * {1,3,7,9,1,3,7,9,1,3}{_})), 10 ), 10 ) = ListaCyfr{10} 
in 
    Wynik
```

## Przykład 2: Tworzenie kalendarza

Deklaracja funkcji generującej tabelę z opcjonalnym argumentem języka:

```powerquery
(DataPoczątkowa as date, DataKońcowa as date, optional Język as nullable text) as table =>
```

Obliczanie liczby dni, generowanie listy dat i konwersja na tabelę:

```powerquery
LiczbaDni = Duration.Days(Duration.From(DataKońcowa - DataPoczątkowa)) + 1,
ListaDat = List.Dates(DataPoczątkowa, LiczbaDni, #duration(1,0,0,0)),
TabelaZListy = Table.FromList(ListaDat, Splitter.SplitByNothing()),
TypZmieniony = Table.TransformColumnTypes(TabelaZListy, {{"Column1", type date}}),
KolumnaData = Table.RenameColumns(TypZmieniony, {{"Column1", "Data"}})
```

Dodawanie kolumn opisowych (rok, miesiąc, nazwa dnia) z uwzględnieniem lokalizacji:

```powerquery
DodanyRok = Table.AddColumn(KolumnaData, "Rok", each Date.Year([Data]), Int16.Type),
DodanyMiesiąc = Table.AddColumn(DodanyRok, "Miesiąc", each Date.Month([Data]), Int8.Type),
DodanaNazwaMiesiąca = Table.AddColumn(DodanyMiesiąc, "NazwaMiesiąca", each Date.ToText([Data], "MMMM", Język), type text),
DodanyNrDniaTygodnia = Table.AddColumn(DodanaNazwaMiesiąca, "NrDniaTygodnia", each Date.DayOfWeek([Data], Day.Monday) + 1, Int8.Type),
DodanaNazwaDniaTygodnia = Table.AddColumn(DodanyNrDniaTygodnia, "NazwaDniaTygodnia", each Date.ToText([Data], "dddd", Język), type text)
```

## Przykład 3: Zamiana wielu wartości (List.Accumulate)

Pobranie tabeli mapowania zmian z bieżącego skoroszytu:

```powerquery
= Excel.CurrentWorkbook(){[Name="FromTo"]}[Content]
```

Konwersja tabeli zmian na listę rekordów (wymagane dla List.Accumulate):

```powerquery
= Table.ToRecords(TabelaZmian)
```

Dynamiczna zamiana wielu fraz w kolumnie tekstowej:

```powerquery
= Table.AddColumn(Źródło, "Opis po zmianach", each List.Accumulate(Rekordy, [Opis], (tekst, rekord) => Text.Replace(tekst, rekord[Znajdź], rekord[Zamień na])), type text)
```

Przykład do analizy funkcji List.Accumulate:

```powerquery
= List.Accumulate({2,4,8}, 1, (s, x) => s + x)
```

## Przykład 4: Poziom w hierarchii (funkcja rekurencyjna)

Pełny kod funkcji `PoziomHierarchii` wywołującej samą siebie:

```powerquery
(IdPracownika as number, TabelaPracowników as table) as number => 
let 
    WierszOsoby = Table.SelectRows(TabelaPracowników, each [ID] = IdPracownika){0}, 
    OsobaPowyżej = WierszOsoby[Przełożony], 
    Poziom = if OsobaPowyżej = null then 0 else @PoziomHierarchii(OsobaPowyżej, TabelaPracowników) + 1 
in 
    Poziom
```
