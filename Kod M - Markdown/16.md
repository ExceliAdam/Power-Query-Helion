# 16. Przykłady użycia Power Query

## Przykład 1: Czy klient powracający

Wyszukiwanie poprzedniego zakupu (funkcja filtrująca):
```powerquery
Table.SelectRows(#"Posortowano wiersze", ((r) => r[Id Klienta] = [Id Klienta] and r[Data Zakupu] < [Data Zakupu]))
```

Wyciągnięcie daty pierwszego (najnowszego) zakupu z przefiltrowanej tabeli:
```powerquery
Table.SelectRows(#"Posortowano wiersze", ((r) => r[Id Klienta] = [Id Klienta] and r[Data Zakupu] < [Data Zakupu]))[Data Zakupu]{0}
```

Sprawdzenie warunku powracającego klienta z obsługą błędów:
```powerquery
try [Odejmowanie] <= 14 otherwise false
```

## Przykład 2: Usuwanie pustych kolumn

Pobranie listy nazw wszystkich kolumn:
```powerquery
= Table.ColumnNames(#"Zmieniono typ")
```

Usuwanie wartości null z konkretnej kolumny (etap pośredni):
```powerquery
= List.RemoveMatchingItems(#"Zmieniono typ"[Column2], {null})
```

Sprawdzenie, czy kolumna jest pusta po usunięciu null i pustych tekstów:
```powerquery
= List.IsEmpty(List.RemoveMatchingItems(#"Zmieniono typ"[Column2],{null, ""}))
```

Dynamiczne wyznaczanie listy kolumn do usunięcia:
```powerquery
= List.Select(Table.ColumnNames(#"Zmieniono typ"), each List.IsEmpty(List.RemoveMatchingItems(Table.Column(#"Zmieniono typ", _),{null, ""})))
```

Pełna formuła usuwająca wszystkie puste kolumny:
```powerquery
= Table.RemoveColumns(#"Zmieniono typ", List.Select(Table.ColumnNames(#"Zmieniono typ"), each List.IsEmpty(List.RemoveMatchingItems(Table.Column(#"Zmieniono typ", _),{null, ""}))))
```

## Przykład 3: Najdłuższe ciągi

Podstawowe grupowanie:
```powerquery
= Table.Group(#"Anulowano przestawienie innych kolumn", {"Imię", "Wartość"}, {{"Liczność", each Table.RowCount(_), Int64.Type}, {"Kolumny", each List.Sum([Atrybut]), type text}})
```

Grupowanie lokalne z łączeniem tekstu i liczeniem wierszy:
```powerquery
= Table.Group(#"Anulowano przestawienie innych kolumn", {"Imię", "Wartość"}, {{"Liczność", each Table.RowCount(_), Int64.Type}, {"Kolumny", each Text.Combine([Atrybut], ", "), type text}}, GroupKind.Local)
```

Znalezienie maksymalnej długości ciągu dla każdego pracownika:
```powerquery
List.Max(Table.SelectRows(#"Przefiltrowano wiersze", (r) => r[Imię] = [Imię])[Liczność])
```

Filtrowanie wierszy odpowiadających maksymalnemu ciągowi:
```powerquery
= Table.SelectRows(#"Dodano kolumnę niestandardową", each ([Liczność] = [Max]))
```
