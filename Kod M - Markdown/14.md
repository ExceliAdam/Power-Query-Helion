# 14. Funkcja z zapytania z parametrem

## Import i łączenie danych z plików .xml

Ścieżka pliku Excel:

```excel
=LEWY(KOMÓRKA("nazwa_pliku");ZNAJDŹ("[";KOMÓRKA("nazwa_pliku"))-1)
```

Nakładanie filtru by wybrać odpowiednią tabelę:

```powerquery
= Table.SelectRows(Źródło, each ([Name] = "Nagłówek"))
```

Rozwinięcie danych z kolumny Table:

```powerquery
= Table.ExpandTableColumn(#"Przefiltrowano wiersze", "Table", {"NumerFaktury", "Data", "Sprzedawca", "Kupujący"}, {"NumerFaktury", "Data", "Sprzedawca", "Kupujący"})
```

Rozwinięcie danych o pozycjach na fakturze:

```powerquery
= Table.TransformColumnTypes(#"Rozwinięty element Pozycja",{{"Opis", type text}, {"Ilość", Int64.Type}, {"VAT", Percentage.Type}})
```

Zmiana typu danych uwzględniająca ustawienia regionalne:

```powerquery
= Table.TransformColumnTypes(#"Zmieniono typ1", {{"Cena", type number}}, "en-US")
```

Scalanie kroków tego samego zapytania:

```powerquery
= Table.NestedJoin(KoniecPrzekształceńPozycji, {"Indeks"}, KoniecPrzekształceńPozycji, {"Indeks"}, "KoniecPrzekształceńPozycji", JoinKind.LeftOuter)
```

## Parametryzacja zapytania

Podstawienie parametru FilePath w miejsce sztywnej ścieżki do pliku:

```powerquery
= Xml.Tables(File.Contents(FilePath))
```

Wywołanie funkcji niestandardowej dla listy plików znajdujących się w folderze:

```powerquery
= Table.AddColumn(#"Usunięto inne kolumny", "XML", each PrzekształćXML([Pełna ścieżka]))
```
