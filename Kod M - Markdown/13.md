# 13. Praca z folderami i wieloma plikami

## Przykład 1: Łączenie danych z plików .csv

Filtrowanie plików w celu pozostawienia jedynie tych z rozszerzeniem .csv:

```powerquery
= Table.SelectRows(#"Tekst pisany małymi literami", each ([Extension] = ".csv"))
```

Wyodrębnianie pierwszego pliku:

```powerquery
= #"Wyodrębniony tekst przed ogranicznikiem"{0}[Content]
```

Odfiltrowanie ukrytych plików:

```powerquery
= Table.SelectRows(#"Wyodrębniony tekst przed ogranicznikiem", each [Attributes]?[Hidden]? <> true)
```

Wywołanie funkcji niestandardowej:

```powerquery
= Table.AddColumn(#"Filtrowane pliki ukryte1", "Przekształć plik", each #"Przekształć plik"([Content]))
```

Usunięcie innych kolumn:

```powerquery
= Table.SelectColumns(#"Wywołaj funkcję niestandardową1", {"Name", "Przekształć plik"})
```


## Przykład 3: Łączenie danych z plików .pdf

Zmiana nazw kolumn:

```powerquery
= Table.RenameColumns(#"Wywołaj funkcję niestandardową", {"Name", "Source.Name"})
```

Usuwanie innych kolumn:

```powerquery
= Table.SelectColumns(#"Wywołaj funkcję niestandardową", {"Przekształć plik"})
```
