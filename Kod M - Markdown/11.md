# 11. Zmiana struktury danych

## Anulowanie przestawienia danych

Usunięcie kolumny Suma przed operacją anulowania przestawienia:
```powerquery
= Table.RemoveColumns(Źródło,{"Suma"})
```

Anulowanie przestawienia kolumn z latami (wybierając te kolumny):
```powerquery
= Table.Unpivot(#"Usunięto kolumny", {"2023", "2024", "2025"}, "Atrybut", "Wartość")
```

Anulowanie przestawienia kolumn z latami (kolumny inne niż Województwo):
```powerquery
= Table.UnpivotOtherColumns(#"Usunięto kolumny", {"Województwo"}, "Atrybut", "Wartość")
```

Zmiana nazw automatycznie utworzonych kolumn:
```powerquery
= Table.RenameColumns(#"Anulowano przestawienie innych kolumn",{{"Atrybut", "Rok"}, {"Wartość", "Przychód"}})
```

## Kolumna przestawna

Przestawienie kolumny bez agregacji (opcja „Nie agreguj”):
```powerquery
= Table.Pivot(#"Zmieniono typ", List.Distinct(#"Zmieniono typ"[Dana]), "Dana", "Wartość")
```

## Połączone informacje

Anulowanie przestawienia innych kolumn ze zmianą nazwy nowej kolumnu:
```powerquery
= Table.UnpivotOtherColumns(#"Zmieniono typ", {"Inwestycja"}, "INWESTYCJA", "Wartość")
```

Podział według ogranicznika:
```powerquery
= Table.SplitColumn(#"Anulowano przestawienie innych kolumn", "Inwestycja", Splitter.SplitTextByEachDelimiter({" "}, QuoteStyle.Csv, false), {"Inwestycja.1", "Okres"})
```

Przestawienie kolumny Inwestycja.1 i ich podsumowanie:
```powerquery
= Table.Pivot(#"Podzielono kolumnę według ogranicznika", List.Distinct(#"Podzielono kolumnę według ogranicznika"[Inwestycja.1]), "Inwestycja.1", "Wartość", List.Sum)
```

## Transpozycja

Wykonanie transpozycji całej tabeli (zamiana wierszy z kolumnami):
```powerquery
= Table.Transpose(Źródło)
```

Podniesienie pierwszego wiersza do nagłówków po transpozycji:
```powerquery
= Table.PromoteHeaders(#"Transponowano tabelę", [PromoteAllScalars=true])
```

## Anulowanie przestawienia kolumn

Anulowanie przestawienia kolumn, gdy kolumna Wartość już istnieje
```powerquery
= Table.UnpivotOtherColumns(#"Zmieniono typ", {"Sprzedawca"}, "Atrybut", "Wartość.1")
```

Anulowanie przestawienia wszystkich kolumn
```powerquery
= Table.UnpivotOtherColumns(#"Zmieniono typ", {}, "Atrybut", "Wartość.1")
```

## Indeks i dzielenie całkowite

Dodanie kolumny indeksu zaczynającej się od zera:
```powerquery
= Table.AddIndexColumn(Źródło, "Indeks", 0, 1, Int64.Type)
```

Wykonanie dzielenia całkowitego na kolumnie indeksu (grupowanie po 3 wiersze):
```powerquery
= Table.TransformColumns(#"Dodano indeks", {{"Indeks", each Number.IntegerDivide(_, 3), Int64.Type}})
```

Przestawienie kolumny Atrybut z wykorzystaniem kolumny Indeks jako klucza grupującego:
```powerquery
= Table.Pivot(#"Kolumna używana podczas dzielenia bez reszty", List.Distinct(
#"Kolumna używana podczas dzielenia bez reszty"[Atrybut]), "Atrybut", "Wartość.1")
```
