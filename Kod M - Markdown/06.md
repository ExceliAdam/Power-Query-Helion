# 6. Łączenie i scalanie danych

## Przekształcanie ocen (liczba znaków)

Dołączanie zapytań (jako nowy krok w istniejącym zapytaniu):
```powerquery
= Table.Combine({Styczeń, Luty, Marzec})
```

Obliczanie liczby gwiazdek poprzez różnicę długości tekstu po zamianie znaków:
```powerquery
Text.Length([Ocena]) - Text.Length (Text.Replace([Ocena]," ★ ",""))
```

Uproszczona formuła obliczająca liczbę gwiazdek przy użyciu funkcji usuwania znaków:
```powerquery
5 - Text.Length (Text.Remove([Ocena]," ★ "))
```

## Lewe Scalania

Scalanie po 1 kolumnie:
```powerquery
= Table.NestedJoin(#"Zmieniono typ", {"Produkt"}, tLCena, {"Produkt"}, "tLCena", JoinKind.LeftOuter)
```

Scalanie po 2 kolumnach:
```powerquery
= Table.NestedJoin(#"Zmieniono typ", {"Produkt", "Rodzaj"}, tLCena2, {"Produkt", "Rodzaj"}, "tLCena2", JoinKind.LeftOuter)
```

Agregowanie scalonych danych:
```powerquery
= Table.AggregateTableColumn(#"Scalone zapytania", "Rozmowy", {
    {"IdKonsultanta", List.Count, "Liczba rozmów"}, 
    {"Czas", List.Sum, "Sumaryczny czas rozmów"}, 
    {"Ocena", each List.Average(List.Transform(_, each 5 - Text.Length(Text.Remove(_, "★")))), "Średnia ocena"}
})
```

## Porównanie list

Antyscalanie:
```powerquery
= Table.NestedJoin(#"Zmieniono typ", {"Magazyn"}, Zamówienie, {"Zamówienia"}, "Zamówienie", JoinKind.RightAnti)
```

## Self join

Scalanie zapytania samego ze sobą:
```powerquery
= Table.NestedJoin(#"Zmieniono typ", {"ID_Przełożonego"}, #"Zmieniono typ", {"ID_Pracownika"}, "Zmieniono typ", JoinKind.LeftOuter)
```

## Scalanie przybliżone

Przykład struktury dla scalania rozmytego:
```powerquery
= Table.FuzzyNestedJoin(#"Zmieniono typ", {"Nazwa Firmy"}, BazaKlientów, {"Nazwa Klienta"}, "BazaKlientów", JoinKind.LeftOuter, [IgnoreCase=true, IgnoreSpace=true, NumberOfMatches=1, Threshold=0.5, TransformationTable=Zamiany])
```
