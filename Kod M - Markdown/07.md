# 7. Grupowanie danych

## Podstawowe grupowanie i agregacje

Formuła grupowania z interfejsu użytkownika:
```powerquery
= Table.Group(#"Zmieniono typ", {"Kategoria"}, {{"Liczba transakcji", each Table.RowCount(_), Int64.Type}, {"Średni przychód", each List.Average([Przychód]), type nullable number}, {"Najczęściej sprzedawany", each List.Median([Produkt]), type nullable text}})
```

Skorygowana formuła grupowania:
```powerquery
= Table.Group(#"Zmieniono typ", {"Kategoria"}, {{"Liczba transakcji", each Table.RowCount(_), Int64.Type}, {"Średni przychód", each List.Average([Przychód]), type nullable number}, {"Najczęściej sprzedawany", each List.Modes([Produkt]), type list}})
```

Końcowy wynik grupowania:
```powerquery
= Table.Group(#"Zmieniono typ", {"Kategoria"}, {{"Liczba transakcji", each Table.RowCount(_), Int64.Type}, {"Średni przychód", each List.Average([Przychód]), type nullable number}, {"Najczęściej sprzedawany", each Text.Combine(List.Modes([Produkt]), ", "), type text}})
```

## Zaawansowane grupowanie (Wszystkie wiersze)

Grupowanie po dwóch kolumnach:
```powerquery
= Table.Group(#"Zmieniono typ", {"Kategoria"}, {{"Przychód kategorii", each List.Sum([Przychód]), type number}, {"% Przychód", each List.Sum([Przychód]), type number}})
```

Grupowanie obliczające procent przychodu:
```powerquery
= Table.Group(#"Zmieniono typ", {"Kategoria"}, {{"Przychód kategorii", each List.Sum([Przychód]), type number}, {"% Przychód", each List.Sum([Przychód]) / List.Sum(#"Zmieniono typ"[Przychód]), Percentage.Type}})
```

Grupowanie obliczające procent przychodu % do rodzica w 1 przekształceniu:
```powerquery
{"% Suma Produkt", each List.Sum([Przychód]) / List.Sum(Table.SelectRows(#"Zmieniono typ", (row) => row[Kategoria] = [Kategoria]{0})[Przychód]), Percentage.Type}
```

## Wyznaczanie ciągów i okresów (Sekwencje)

Scalenie zapytania samego ze sobą:
```powerquery
= Table.NestedJoin(#"Pogrupowano wiersze1", {"Kategoria"}, #"Pogrupowano wiersze1", {"Kategoria"}, "Pogrupowano wiersze1", JoinKind.LeftOuter)
```

Dzielenie po rozwinięciu kolumny:
```powerquery
= Table.AddColumn(#"Rozwinięty element Scalone", "% produktu", each [#"% w kategorii"] / [Przychód kategorii], Percentage.Type)
```

Scalenie różnych kroków zapytania:
```powerquery
= Table.NestedJoin(PrzychódProdukty, {"Kategoria"}, PrzychódKategorie, {"Kategoria"}, "Scalone", JoinKind.LeftOuter)
```

## Grupowanie lokalne

Zamiana daty na nazwę miesiąca:
```powerquery
= Table.TransformColumns(#"Zmieniono typ", {{"Data", each Date.MonthName(_), type text}}) number}})
```

Zamiana daty na odpowiednio sformatowany tekst:
```powerquery
= Table.TransformColumns(#"Zmieniono typ", {{"Data", each DateTime.ToText(_, "yyyy-MM-dd[WŻ33.1][AK33.2]"), type text}})
```

Grupowanie tworzące zapis okresu pomiędzy datami:
```powerquery
= Table.Group(#"Wyodrębniono nazwę miesiąca", {"Produkt", "Stan"}, {{"Liczba dni", each Table.RowCount(_), Int64.Type}, {"Okres", each Text.Combine({"Od ", List.Min([Data]), " do ", List.Max([Data])}), type text}}, GroupKind.Local)
```
