# 5. Praca z liczbami, datami i czasem

## Statystyczne podsumowania i obliczenia

Średnia z całej kolumny:
```powerquery
= List.Average(#"Zmieniono typ"[Sty])
```

Obliczanie stosunku wartości do średniej z kolumny (po wcześniejszym wyodrębnieniu średniej jako skalar `ŚredniaSty`):
```powerquery
= Table.AddColumn(#"Zmieniono typ", "% do średniej", each [Sty]/ŚredniaSty)
```

Obliczanie średniej z wielu kolumn w jednym wierszu przy użyciu listy:
```powerquery
= Table.AddColumn(#"Zmieniono typ", "Średnia", each List.Average({[Sty], [Lut], [Mar]}), type number)
```

Zwiększenie wartości o 15%:
```powerquery
= Table.AddColumn(#"Wstawiono średnią", "Wartość procentowa", each [Mar] * 1.15, type number)
```

% stosunek wartości z różnych miesięcy:
```powerquery
= Table.AddColumn(#"Wstawiona wartość procentowa", "Wartość procentowa.1", each [Mar] / [Lut] * 100, Percentage.Type)
```

## Zaokrąglanie liczb

Standardowe zaokrąglenie do groszy:
```powerquery
= Table.AddColumn(#"Zmieniono typ", "Zaokrąglenie", each Number.Round([Stawka za 100], 2), type number)
```

Zaokrąglanie do pełnych setek w dół (użycie parametru ujemnego w funkcji):
```powerquery
= Table.TransformColumns(#"Wstawiono mnożenie", {{"Suma znaków", each Number.RoundDown(_, -2), Int64.Type}})
```

## Przekształcanie dat i czasu

Wyciąganie nazwy dnia tygodnia:
```powerquery
Date.DayOfWeekName([Data])
```

Zmiana języka nazw dni tygodnia poprzez drugi argument funkcji:
```powerquery
Date.DayOfWeekName([Data], "pl-PL")
```

Obliczanie różnicy między datami (wynik w formacie `duration`):
```powerquery
= Table.AddColumn(#"Zmieniono typ", "Czas trwania", each [Data końcowa] - [Data początkowa], type duration)
```

Konwersja czasu trwania na całkowitą liczbę dni:
```powerquery
Duration.TotalDays([Data dzisiejsza] - [Data zatrudnienia])
```

Obliczanie przybliżonego stażu pracy w latach z zaokrągleniem w dół:
```powerquery
Number.RoundDown(Duration.TotalDays(DateTime.LocalNow() - [Data zatrudnienia]) / 365.25)
```

## Praca z godzinami

Obliczanie różnicy czasu (godzina wyjścia minus godzina wejścia):
```powerquery
= Table.AddColumn(#"Zmieniono typ", "Czas pracy", each [Godzina wyjścia] - [Godzina wejścia], type duration)
```

Obsługa przejścia przez północ (dodanie 1 dnia do różnicy, jeśli wynik jest ujemny):
```powerquery
= Table.AddColumn(#"Dodano kolumnę niestandardową", "Czas pracy", each if [Godzina wyjścia] < [Godzina wejścia] then ([Godzina wyjścia] - [Godzina wejścia]) + #duration(1, 0, 0, 0) else [Godzina wyjścia] - [Godzina wejścia], type duration)
```

## Kolumna z przykładów

Proste scalenie wartości:
```powerquery
Text.Combine({Text.Proper([Imię]), " K."})
```

Skomplikowane scalenie przekształconych wartości:
Proste scalenie wartości:
```powerquery
= Table.AddColumn(#"Zmieniono typ", "Rok urodzenia klienta", each Text.Combine({Text.Proper([Imię]), " ", Text.Start(Text.Upper([Nazwisko]), 1), ". rok urodzenia ", DateTime.ToText([Data urodzenia], "yyyy")}), type text)
```

Mnożenie z zaokrągleniem:
```powerquery
= Table.AddColumn(#"Zmieniono typ", "Przychód", each Number.Round([Ilość] * [#"Cena jedn."],2), type number)
```
